Preguntas Cuestionario 1

¿Qué es entonces una base de datos - DB?
Conjunto de datos pertennecientes a un mismo contexto, almacenados semánticamente para ser usados posteriormente.

¿En qué casos es conveniente usar bases de datos?
Cuando tienes grandes cantidades de informaciín que tienen que ser ordenadas para posteriormente ser visualziadas.

¿Qué es un sistema de gestión de base de datos - DBMS?
Software que administra los datos en una base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.
Controlar la organización, almacenamiento, recuperación eintegridad de los datos contenidos en una base dde datos.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
Me hubiera gusta haber implementado una base datos por mi mismo en proyecto CAD, ya que habí mucha informaciín proveniente de muchos sensores
trabajando en diversas máquinas. De haber almacenado esta información, se hbaría podido hacer un analísis mas efeiciente de los problemas
de producción de las empresas.
